class LowerZonePersistenceService {
  static const String _dawKey = 'lower_zone_daw_state';
  static const String _middlewareKey = 'lower_zone_middleware_state';
  static const String _slotLabKey = 'lower_zone_slotlab_state';

  // Singleton
  static LowerZonePersistenceService get instance;

  // Per-section load/save
  Future<DawLowerZoneState> loadDawState();
  Future<bool> saveDawState(DawLowerZoneState state);
  // ... similar for Middleware and SlotLab

  // Batch operations
  Future<({DawLowerZoneState daw, MiddlewareLowerZoneState middleware, SlotLabLowerZoneState slotLab})> loadAllStates();
  Future<void> clearAllStates();

### 14.3 Controller Auto-Save

Svi controlleri koriste `_updateAndSave()` pattern:

```dart
void _updateAndSave(DawLowerZoneState newState) {
  _state = newState;
  notifyListeners();
  saveToStorage();  // Async, non-blocking
}

// Koristi se u svim metodama koje menjaju state:
void setSuperTab(DawSuperTab tab) {
  _updateAndSave(_state.copyWith(superTab: tab, isExpanded: true));
}
```

### 14.4 Initialization

U `main.dart`:

```dart
void main() async {
  WidgetsFlutterBinding.ensureInitialized();
  await LowerZonePersistenceService.instance.init();
  runApp(const FluxForgeApp());

```

Controlleri mogu load-ovati state u `initState`:

```dart
@override
void initState() {
  super.initState();

}
```

### 14.5 What Gets Persisted

Per section:
- Current super-tab
- Current sub-tab for each super-tab
- Expand/collapse state
- Panel height

**NOT persisted** (transient state):
- Scroll positions within panels
- Selection state
- Search filters
- Temporary UI state
